# 🏦 Pro Loan Predictor

A robust Machine Learning application built using Python, scikit-learn, and Streamlit. This app analyzes financial parameters (like income, credit history, and loan term) using a trained Logistic Regression model to predict loan approval probability with a responsive UI.

## 💻 Tech Stack
* **UI Framework:** Streamlit
* **Machine Learning:** Scikit-Learn
* **Data Processing:** Pandas, NumPy
* **Language:** Python
